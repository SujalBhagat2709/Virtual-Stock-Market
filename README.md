# Virtual Stock Market

## Overview

A simple stock market simulation where buying and selling shares affects stock prices dynamically.

---

## Files

- market.py
- trading.py

---

## Run

```bash
python trading.py
```

---

## Commands

Buy Shares:

```text
buy TechX 100
```

Sell Shares:

```text
sell TechX 50
```

Exit:

```text
exit
```

---

## Example

Initial Market:

```text
TechX ₹100
GameCorp ₹50
FoodHub ₹75
```

Command:

```text
buy TechX 100
```

Output:

```text
New Price: ₹110
```

Command:

```text
sell TechX 50
```

Output:

```text
New Price: ₹105
```
